background 속성

  • background-color : 배경을 단색으로 설정 -> 색상명, rgb코드
  • background-image : 배경을 이미지로 설정 -> url(이미지 파일명)
  • background-repeat : 배경 이미지 반복 여부 설정 -> no-repeat, repeat-x(가로방향반복), repeat-y(세로방향반복)
  • background-position : 배경이미지의 위치 설정 -> left, center, right, top, middle, bottom, %
  • background-attachment : 배경이미지 고정 설정 -> fixed, scroll
  • background-size : 배경이미지 크기 설정
